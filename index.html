Git Rebase — это мощный инструмент в Git, который позволяет изменять историю коммитов путем перемещения вашей текущей ветки на другой коммит

\

Команда git fetch загружает коммиты, файлы и ссылки из удаленного репозитория в ваш локальный репозиторий.

Команда git pull — это автоматизированная версия команды git fetch.

Команда git merge объединяет несколько последовательностей коммитов в общую историю.

Что собой представляет Git-flow? Git-flow — альтернативная модель ветвления Git, в которой используются функциональные ветки и несколько основных веток.
